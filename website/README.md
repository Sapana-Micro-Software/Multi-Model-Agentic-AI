# Multi-Model Agentic AI Website

Modern Next.js website with TypeScript, Tailwind CSS, Canvas, and SVG visualizations.

## Features

- ✅ Next.js 14 with App Router
- ✅ TypeScript for type safety
- ✅ Tailwind CSS for styling
- ✅ Canvas-based architecture diagrams
- ✅ SVG charts and graphs (Recharts)
- ✅ Interactive comparison matrices
- ✅ Framer Motion animations
- ✅ Responsive design
- ✅ Performance benchmarks visualization

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

The static site will be generated in the `out` directory.

## Components

- `Hero` - Animated hero section with particles
- `Stats` - Statistics cards
- `Features` - Feature grid
- `ArchitectureDiagram` - Canvas-based system architecture
- `Benchmarks` - Performance charts (Recharts)
- `ComparisonMatrix` - Interactive feature comparison table
- `Studies` - Research studies grid
- `PaperSection` - Paper download section
- `CodeSection` - Code repository links

## Deployment

The site can be deployed to:
- Vercel (recommended for Next.js)
- GitHub Pages (static export)
- Any static hosting service
